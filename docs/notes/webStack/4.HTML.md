## 简介

HTML（HyperText Markup Language）是用于创建网页的标准标记语言。

## 基本结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
</head>
<body>
    <h1>欢迎来到我的网站</h1>
    <p>这是一个段落。</p>
</body>
</html>
```

## 常用标签

### 文本标签
- `<h1>` - `<h6>`: 标题
- `<p>`: 段落
- `<span>`: 行内元素
- `<div>`: 块级元素

### 链接和媒体
- `<a href="">`: 链接
- `<img src="" alt="">`: 图片
- `<video>`: 视频
- `<audio>`: 音频

### 列表
- `<ul>`: 无序列表
- `<ol>`: 有序列表
- `<li>`: 列表项

### 表格
- `<table>`: 表格
- `<tr>`: 表格行
- `<td>`: 表格单元格
- `<th>`: 表格标题单元格
