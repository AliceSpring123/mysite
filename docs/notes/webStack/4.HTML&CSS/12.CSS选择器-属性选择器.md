属性选择器是根据元素的属性特征来精准定位元素，从而实现更灵活的样式控制

语法是`[]`:

| 属性选择器 | 作用                               |
| ---------- | ---------------------------------- |
| [属性]     | 匹配包含指定属性的元素             |
| [属性=值]  | 匹配属性值等于指定值的元素         |
| [属性^=值] | 匹配属性值以指定字符串开头的元素   |
| [属性$=值] | 匹配属性值以指定字符串结尾的元素   |
| [属性*=值] | 匹配属性任意位置包含指定子串的元素 |


示例:

```css

div[class*="box"] {
  width: 200px;
  height: 200px;
  margin: 20px;
}

```
