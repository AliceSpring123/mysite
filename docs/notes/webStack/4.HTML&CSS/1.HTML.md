## 简介

HTML（HyperText Markup Language）是用于创建网页的标准标记语言。

## 基本结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>页面标题</title>
    </head>
    <body>
        <h1>欢迎来到我的网站</h1>
        <p>这是一个段落。</p>
    </body>
</html>
```

## DOCTYPE声明
DOCTYPE是 `document type` 的缩写。 `<!DOCTYPE html>`是 HTML5 的声明，位于文档的最前面，处于标签之前。

这是网页必备的组成部分，避免了浏览器的怪异模式。

## html

定义 HTML 文档，所有其他的HTML元素都要包裹在 html 标签中

```html
<!DOCTYPE html>
<html lang="zh-CN">
    ...
</html>
```
`lang`用于声明网页的主要语言，帮助浏览器、搜索引擎等正确处理页面内容。
- `en` 代表英语
- `zh-CN` 代表中文

## head

head标签用于定义文档的头部，文档的头部描述了文档的各种属性和信息，包括文档的标题、在 web 中的位置以及和其他文档的关系等。绝大多数文档头部包含的数据都不会作为内容展示。

```html
<!DOCTYPE html>
<html>
    <head>...</head>
</html>
```

head标签中包含的标签有：

### title 标签

`<title>` 是 `<head>` 标签中唯一必要的元素，它定义文档的标题，显示在浏览器窗口的标题栏或状态栏上。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>这是标题</title>
    </head>
</html>
```

### meta 标签

meta标签用来描述一个HTML网页文档的属性、关键词等，例如：`charset="utf-8"`，一些常用的meta设置如下：

```html
<!DOCTYPE html>
<html>
    <head>
        <!-- 字符编码声明 -->
        <meta charset="UTF-8">

        <!-- 视口设置，响应式设计必备 -->
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        
        <!-- 网页兼容性设置 -->
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        
        <!-- 网页描述 -->
        <meta name="description" content="这是一个关于HTML head标签详解的示例页面">
        
        <!-- 网页关键词 -->
        <meta name="keywords" content="HTML, head标签, 元数据, SEO, 网页开发">
        
        <!-- 作者信息 -->
        <meta name="author" content="网页开发者">
        
        <!-- 版权信息 -->
        <meta name="copyright" content="版权所有">
    </head>
</html>
```

## body

body 元素定义文档的主体。

body元素包含文档的所有内容，比如文本、超链接、图像、表格和列表等

它会直接在页面中显示出来，也就是用户可以直观看到的内容。

```html
<!DOCTYPE html>
<html>
    <head>
        ...
    </head>

    <body>
        ...
    <body>
</html>
```

### h1~h6 标签

`<h1>` ~ `<h6>` 用来定义正文文档的标题

```html
<h1> 一级标题 <h1>
<h2> 二级标题 <h2>
<h3> 三级标题 <h3>
<h4> 四级标题 <h4>
<h5> 五级标题 <h5>
<h6> 六级标题 <h6>
```

常用属性：

- `id` - 唯一标识符

- `class` - CSS类名

- `style` - 内联样式

- `align` - 对齐方式 (left|center|right)

### 段落和文本标签

- 段落标签

    定义文本段落，浏览器会自动在段落前后添加空白

    ```html
    <p>这是一个段落</p>
    ```

    常用属性：

    - `id` - 唯一标识符
    - `class` - CSS类名
    - `style` - 内联样式
    - `align` - 对齐方式

- 文本标签

    用于格式化文本的语义化标签

    ```html
    <span id="name" class="highlight" style="color: red;">行内文本容器</span>
    <strong title="重要内容" class="emphasis">强调文本</strong>
    <em class="italic-text">斜体文本</em>
    <mark class="yellow-bg">高亮文本</mark>
    <b>加粗文本</b>
    <del>文本删除线</del>
    <small>小号文本</small>
    <sub>下标</sub>
    <sup>上标</sup>
    ```

    常用属性：

    - `id` - 唯一标识符
    - `class` - CSS类名
    - `style` - 内联样式
    - `title` - 提示文本


### 链接标签

创建超链接，用于跳转到其他页面、位置或执行操作

```html
<a href="https://example.com" target="_blank" rel="noopener" 
   id="link1" class="external-link" title="访问示例网站"
   download="filename.pdf">
   链接文本
</a>
```

常用属性：

- `href` - 链接地址 (URL或锚点)

- `target` - 打开方式 (_blank|_self|_parent|_top)

- `rel` - 关系 (noopener|noreferrer|external)

- `id` - 唯一标识符

- `class` - CSS类名

- `title` - 提示文本

- `download` - 下载文件名

### 引用标签

```html
<blockquote>块级引用</blockquote>
<q>行内引用</q>
<cite>引用标题</cite>
```

### 图片标签

在文档中嵌入图像，必须提供替代文本

```html
<img src="image.jpg" alt="图片描述" width="300" height="200"
     id="banner" class="responsive-img" title="图片标题"
     loading="lazy" srcset="image-2x.jpg 2x" sizes="(max-width: 600px) 100vw">
```

常用属性：

- `src` - 图片路径

- `alt` - 替代文本 (必需)

- `width` - 宽度

- `height` - 高度

- `id` - 唯一标识符

- `class` - CSS类名

- `title` - 提示文本

- `loading` - 加载方式 (lazy|eager)

- `srcset` - 响应式图片源

- `sizes` - 响应式尺寸

### 有序列表

创建有顺序编号的项目列表

```html
<ol id="steps" class="procedure" type="1" start="1" reversed>
  <li>第一步</li>
  <li>第二步</li>
</ol>
```

`<ol>`标签的`type`属性表示排序时使用的标号：

- `"1"` - 使用数字符号（1，2，3）
- `"a"` - 使用小写字母标号（a，b，c）
- `"A"` - 使用大写字母标号（A，B，C）
- `"i"` - 使用小写罗马数字标号（i，ii，iii）
- `"I"` - 使用大写罗马数字标号（I，II，III）

### 无序列表

创建没有特定顺序的项目列表

```html
<ul id="nav-list" class="menu" type="disc">
  <li id="item1" class="menu-item">列表项1</li>
  <li id="item2" class="menu-item">列表项2</li>
</ul>
```

`<ol>`标签的`type`属性表示列表标号：

- `"disc"` - 实心圆（默认）
- `"circle"` - 空心圆
- `"square"` - 小方块
- `"none"` - 不显示

### 定义列表

创建术语及其定义的列表

```html
<dl id="glossary" class="definition-list">
  <dt id="term1">HTML</dt>
  <dd>超文本标记语言</dd>
  <dt id="term2">CSS</dt>
  <dd>层叠样式表</dd>
</dl>
```

### 表格

完整表格结构：

```html
<table id="student-table" class="data-table" border="1">
  <!-- 表格标题 -->
  <caption>学生成绩表</caption>
  
  <!-- 表头 -->
  <thead>
    <tr>
      <th>姓名</th>
      <th>年龄</th>
      <th>成绩</th>
    </tr>
  </thead>
  
  <!-- 表格主体 -->
  <tbody>
    <tr>
      <td>张三</td>
      <td>20</td>
      <td>85</td>
    </tr>
    <tr>
      <td>李四</td>
      <td>19</td>
      <td>92</td>
    </tr>
  </tbody>
  
  <!-- 表脚 -->
  <tfoot>
    <tr>
      <td>平均</td>
      <td>19.5</td>
      <td>88.5</td>
    </tr>

```

#### `<table>` 标签

定义表格的容器

属性：

- `border` - 表格边框宽度

- `cellpadding` - 单元格内边距

- `cellspacing` - 单元格间距

- `width` - 表格宽度

- `height` - 表格高度

- `align` - 表格对齐方式 (`left`|`center`|`right`)

- `bgcolor` - 表格背景颜色

- `summary` - 表格摘要（用于可访问性）

#### `<caption>` 标签

定义表格标题

属性：

- `align` - 标题对齐方式 (`top`|`bottom`)

```html
<caption align="top">2023年学生成绩统计表</caption>
```

#### `<thead>` 标签

定义表格的表头部分，包含列标题

属性：

- `align` - 水平对齐

- `valign` - 垂直对齐

- `bgcolor` - 背景颜色


#### `<tbody>` 标签

定义表格的主体内容

属性：

- `align` - 水平对齐

- `valign` - 垂直对齐

#### `<tfoot>` 标签

定义表格的页脚，通常包含汇总信息

```html
<tfoot>
  <!-- 汇总行 -->
</tfoot>
```

#### `<th>` 标签

定义表头单元格，通常用于列标题

属性：

- `scope` - 关联范围 (`col`|`row`|`colgroup`|`rowgroup`)

- `colspan` - 横跨列数

- `rowspan` - 横跨行数

- `abbr` - 缩写形式

- `headers` - 关联的表头ID

- `align` - 水平对齐

- `valign` - 垂直对齐

- `width` - 单元格宽度

- `height` - 单元格高度

- `bgcolor` - 背景颜色

#### `<tr>`标签

定义表格中的行

属性：

- `align` - 行内单元格水平对齐

- `valign` - 行内单元格垂直对齐

- `bgcolor` - 行背景颜色

- `height` - 行高度

#### `<td>` 标签

定义标准单元格

属性：

- `colspan` - 横跨列数

- `rowspan` - 横跨行数

- `headers` - 关联的表头ID

- `align` - 水平对齐

- `valign` - 垂直对齐

- `width` - 单元格宽度

- `height` - 单元格高度

- `bgcolor` - 背景颜色

- `nowrap` - 禁止换行

### 表单

#### `<form>`标签

创建包含表单元素的容器，用于收集用户输入

```html
<form action="/submit" method="post" enctype="multipart/form-data"
      autocomplete="on" target="_self" novalidate name="userForm">
  <!-- 表单元素 -->
</form>
```


属性：

- `action` - 表单提交的服务器地址

- `method` - 提交方法 (`get`|`post`)

- `target` - 响应打开位置 (`_blank`|`_self`|`_parent`|`_top`)

- `enctype` - 编码类型 (`application/x-www-form-urlencoded`|`multipart/form-data`|`text/plain`)

- `autocomplete` - 自动完成 (`on`|`off`)

- `novalidate` - 不验证表单

- `name` - 表单名称

- `accept-charset` - 字符编码


#### 表单元素-文本框

文本域通过 `<input type="text">` 标签来设定，当用户要在表单中键入字母、数字等内容时使用

```html
<form>
    First name: <input typt="text", name="firstname">
    <br>
    Last name: <input typt="text", name="lastname">
</form>
```

#### 表单元素-密码框

文本域通过 `<input type="password">` 标签来设定

```html
<form>
    Password: <input typt="password", name="pwd">
</form>
```

#### 表单元素-提交按钮

表单的`action`属性定义了目标文件名。当用户点击确认按钮时，表单的内容会被传送到目标文件，在这个目标文件中进行数据处理。

```html
<form name="input", action="url", method="get">
    Username: <input type="text" name="user">
    <input type="submit", value="登录">
</form>
```

## 交互组件

### 按钮

按钮是用户触发操作的主要交互元素，用于提交表单、执行命令等。

```html
<button type="button" onclick="alert('按钮被点击了!')">点击我</button>
<button type="submit">提交表单</button>
<button type="reset">重置</button>
```

常用属性说明:

- `type` - 按钮类型：`button`(普通)、`submit`(提交)、`reset`(重置)

- `disabled` - 禁用按钮

- `autofocus` - 页面加载时自动聚焦

- `form` - 关联的表单ID

- `name` - 按钮名称

- `value` - 按钮值

### 文本输入框

允许用户输入单行文本的基本表单控件。

```html
<label for="username">用户名：</label>
<input type="text" id="username" name="username" placeholder="请输入用户名">
```

常用属性说明
- `type="text"` - 输入类型为文本

- `placeholder` - 提示文本

- `maxlength` - 最大字符数

- `minlength` - 最小字符数

- `required` - 必填字段

- `readonly` - 只读

- `disabled` - 禁用

- `pattern` - 正则验证模式

### 复选框

允许用户从多个选项中选择一个或多个选项。

```html
<label><input type="checkbox" name="hobby" value="reading"> 阅读</label>
```

常用属性说明

- `type="checkbox"` - 输入类型为复选框

- `checked` - 默认选中

- `name` - 字段名称

### 单选框

允许用户从多个互斥的选项中选择一个选项。

### 下拉选择框

## 块元素和行内元素

### 块元素

块级元素在页面中以块的形式展现，通常会独占一行，宽度默认填满父容器，可以设置宽度、高度、内外边距等属性。

**主要特性**
- 默认宽度为100%（占满父容器）

- 总是从新行开始

- 可以包含其他块级元素和行内元素

- 可以设置宽度、高度、上下内外边距

**常见块级元素**

`<div>` - 通用块容器

`<h1>`-`<h6>` - 标题

`<p>` - 段落

`<ul>`, `<ol>`, `<li>` - 列表

`<table>` - 表格

`<form>` - 表单

`<header>`, `<footer>`, `<section>`, `<article>`, `<aside>`, `<nav>` - 语义化标签

`<hr>` - 水平线

`<blockquote>` - 块引用

### 行内元素

行内元素不会独占一行，只占据内容所需的宽度，多个行内元素可以在同一行内显示。

**主要特性**
- 宽度由内容决定

- 不会强制换行

- 只能包含文本和其他行内元素

- 设置宽度、高度无效

- 上下外边距无效

**常见行内元素**

- `<span>` - 通用行内容器

- `<a>` - 链接

- `<strong>`, `<em>` - 强调文本

- `<img>` - 图片

- `<input>` - 输入框

- `<button>` - 按钮

- `<label>` - 标签

- `<code>` - 代码

- `<sup>`, `<sub>` - 上标下标

- `<br>` - 换行


### 行内块元素

结合了块级和行内元素的特性，既不会强制换行or占满一行，又可以设置宽度、高度和内外边距。

主要特性
- 不换行：像行内元素一样在同一行显示

- 可设置宽高：像块级元素一样可以定义尺寸

- 边距有效：四个方向的内外边距都有效

- 默认对齐：默认基线对齐，可通过 vertical-align 调整

**示例**

```html
<div style="display: inline-block; border: 2px solid purple; 
            width: 100px; height: 60px; margin: 5px;">
  行内块1
</div>
<div style="display: inline-block; border: 2px solid orange; 
            width: 100px; height: 60px; margin: 5px;">
  行内块2
</div>
```

`style` 属性中的 `display` 可以强制转换块元素为行内元素，或将行内元素转换为块元素

常用`display`值

- `block` - 块级显示

- `inline` - 行内显示

- `inline-block` - 行内块显示

- `none` - 不显示

- `flex` - 弹性盒子

- `grid` - 网格布局


## 通用属性总结

几乎所有HTML标签都支持以下属性：

- `id` - 元素的唯一标识符

- `class` - 元素的CSS类名

- `style` - 元素的内联样式

- `title` - 元素的提示文本

- `lang` - 元素内容的语言

- `dir` - 文本方向 (ltr|rtl)

- `data-*` - 自定义数据属性

- `aria-*` - 可访问性属性

- `hidden` - 隐藏元素

- `tabindex` - 定义Tab键导航顺序

- `accesskey` - 定义快捷键

- `contenteditable` - 是否可编辑

- `draggable` - 是否可拖拽

- `spellcheck` - 是否进行拼写检查

这些标签和属性构成了HTML的基础，合理使用它们可以创建结构良好、语义清晰且易于访问的网页。
